# Hackpad---macropad
I want to build a hackpad or macropad that i can use when watching yt videos or social media


websites used to make the layout of buttons:
https://kbplate.ai03.com/
https://www.keyboard-layout-editor.com/#/


BOM - Components used:
- 4 keys
- 1 xiao rp2040-DIP
- 1 oled panel
- 4 sk6812MINI LEDs
